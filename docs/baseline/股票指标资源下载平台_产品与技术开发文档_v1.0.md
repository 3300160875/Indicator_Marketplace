# 股票指标资源下载平台：产品需求、技术架构与研发实施文档

**架构：** WordPress + Easy Digital Downloads + 自研资源/人工支付/私有下载/运营插件  
**版本：** V1.0  
**日期：** 2026-06-24  

> 本文件为 DOCX 正式文档的便携源稿摘要。完整页面级功能、字段表、权限矩阵、测试用例与验收清单请以 DOCX 为准。

## 1. 产品定位

面向国内股票技术分析爱好者的内容型数字资源平台。核心链路为：内容发现 → 资源判断 → 免费/单品/VIP 权益 → 人工付款核验 → 私有下载 → 安装与售后。平台不提供个股买卖建议，不承诺收益，不经营破解或无授权资源。

## 2. 首期架构

- WordPress 7.x、PHP 8.3+、MariaDB 10.11 LTS、Nginx、Redis。
- Easy Digital Downloads 3.x 作为订单与客户事实来源。
- 建议采购 EDD All Access，并通过 EntitlementService 封装。
- 自定义服务端渲染主题，不做 SPA。
- 四个插件：`sr-core`、`sr-manual-payment`、`sr-private-download`、`sr-admin-ops`。
- 私有 COS/OSS/S3；下载前验证权益与配额，再生成 120 秒短时签名 URL。

## 3. 首期支付

普通个人收款码不具备标准网站支付回调能力。首期只设计“订单 → 用户提交凭证 → 财务核对真实账单 → 批准/驳回 → EDD 完成订单 → 授予权益”的人工流程，不做通知监听、Cookie 读取、挂机猜单或来源不明的免签支付。生产环境默认关闭，Gate 0 通过后才能启用。

## 4. P0 模块

1. 首页、分类、标签、筛选、搜索、详情、教程、VIP 页面。
2. 注册登录、订单、用户中心、下载中心、工单。
3. 资源模型、属性、版本、文件、版权证据与发布审核。
4. EDD 单品购买、订单快照、人工付款凭证与审核。
5. VIP 套餐、范围、期限、排除项与每日配额。
6. 私有存储、一次性令牌、防直链、下载日志与异常控制。
7. 运营后台、合规审核、客服、报表、审计与配置。
8. CI/CD、缓存、监控、备份、恢复、安全与性能。

## 6. 核心验收

- P0 测试 100% 通过，无阻断/严重缺陷。
- 完成订单必须一次且仅一次获得正确权益。
- 跨用户订单、付款凭证、工单和下载均不可越权。
- 配额并发原子，令牌不可重放，私有文件不可直链。
- 下载成功率 ≥98%，移动 LCP p75 ≤2.5 秒。
- RPO ≤1 小时，RTO ≤4 小时，完成恢复演练。
- 所有付费资源具备版权证据并通过证券业务边界/文案审核。

## 7. Gate 0

上线付费前必须确认经营主体与收款方式、证券投资咨询边界、内容版权、消费者规则、隐私、网站部署/备案等要求。未通过时，网站仅上线免费内容，付款功能保持关闭。
