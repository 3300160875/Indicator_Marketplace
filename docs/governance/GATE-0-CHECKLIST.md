
# Gate 0：经营、支付、版权与内容边界

生产支付、付款凭证上传和付费下载默认关闭。以下项目全部有书面结论后方可开启：

- [ ] 经营主体/登记义务与网站备案路径；
- [ ] 个人码人工核验模式是否允许用于当前经营场景；
- [ ] 收款人名称、订单说明、退款和消费者权益规则；
- [ ] 隐私政策、付款凭证保存期限和后台访问角色；
- [ ] 股票指标内容不构成收益承诺、荐股或投资顾问服务；
- [ ] 首批资源来源、授权证据、下架与投诉机制；
- [ ] 发票/税务及账务留痕路径；
- [ ] 正式支付接口迁移条件和关闭人工模式的阈值。

## Feature Flags

- `SR_PAYMENTS_ENABLED=false`
- `SR_MANUAL_PAYMENT_ENABLED=false`
- `SR_PAID_DOWNLOADS_ENABLED=false`
- `SR_UPLOAD_PROOFS_ENABLED=false`

Gate 未通过时只允许公开内容、免费资源和内部测试白名单。
