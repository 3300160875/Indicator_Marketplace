
# Runbook：人工付款审核

1. 按状态领取提交记录，确认未被其他审核员锁定；
2. 查看订单号、应付金额、提交时间、凭证缩略图和已有交易指纹；
3. 在官方账户账单中人工核对真实到账，不以截图单独作为依据；
4. 输入/确认可追踪的交易参考号，系统计算指纹并检查唯一性；
5. 批准前再次锁订单和提交状态；批准动作必须幂等；
6. 系统完成 EDD 订单，权益监听器发放权益；
7. 查看审计记录、订单、权益和通知 Outbox 是否一致；
8. 无法确认时使用 `needs_more_info` 或 `rejected`，禁止“先放行后补证”。

发生重复认领、金额不一致、账户未到账、订单已完成或状态冲突时，停止审批并创建异常工单。
